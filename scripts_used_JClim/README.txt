These are the python scripts used for the analysis of Touma & Deser, 2026, Journal of Climate

## FWI calculation scripts:
FWIsystem.py - functions for calculating moisture codes and fire weather indices and for writing files
FWI_system_var_hist.py - uses FWIsystem.py to calculate mapped FWI
FWI_system_var_hist_mean.py - uses FWIsystem.py to calculate detrended FWI
